# 🏥 Hospital Patient Data Management System

A Python + MySQL project to manage hospital records.

## Features
- Add Doctor & Patient Records
- Hospitalized Patient Management
- Discharge & Bill Calculation
- Rehab Records Tracking
- Password Protected Login (4-attempt lockout)
- Main Menu connecting all modules

## Tech Stack
- Python 3
- MySQL 9.7
- mysql-connector-python

## How to Run
1. Run `hospital_setup.sql` in MySQL Workbench
2. Update DB password in `hospital_management.py`
3. Run: `python3 hospital_management.py`
